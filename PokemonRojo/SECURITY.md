# Política de seguridad

## Información sensible

Este repositorio no debe contener credenciales, ROMs, partidas, estados del
emulador ni rutas privadas. Si detectas ese contenido en una rama pública, no lo
copies a un issue. Informa de forma privada mediante un
[aviso de seguridad de GitHub](https://github.com/Cordovadcrk/PokeTraining/security/advisories/new).

## Vulnerabilidades

Incluye versión de PokeTraining, sistema operativo, versión de Python, pasos
mínimos para reproducir y efecto observado. No adjuntes la ROM: usa únicamente
su huella criptográfica cuando sea relevante.

El proyecto está en fase alfa y no debe usarse como aislamiento de código o
datos no confiables. Las versiones soportadas son las declaradas en
`pyproject.toml`; las dependencias se revisan y actualizan mediante cambios
explícitos y pruebas.

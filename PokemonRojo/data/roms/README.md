# ROM local

Coloca aquí una copia obtenida legalmente de Pokémon Red con el nombre
`POKEMON_RED.gb`.

La ROM, las partidas guardadas y los estados del emulador están excluidos por
`.gitignore`. Este proyecto no distribuye el juego ni proporciona enlaces de
descarga. Las direcciones WRAM documentadas corresponden a Pokémon Red/Blue de
primera generación y deben validarse para la edición concreta que se utilice.
